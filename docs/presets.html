Bulk governance

治理预设

把论文库里的批量状态迁移、复习安排和分类写回动作整理成可读、可校验、可被桌面端复用的 preset 目录。

Presets JSON Schema 论文库 工作流中心 preset 5 workflow 2 active personal

推荐流程

  1. 把常用批量动作写进 docs/guides/taxonomy.json 的 bulk_presets，避免每次在页面里临时拼 CSV。
  2. 当一套 preset 要跨 personal/research/implementation workflow 复用时，优先使用各 workflow 都存在的候选值。
  3. DMG 或桌面端可以读取 presets.json，把 preset 渲染为菜单项，再把 patch_columns 映射成批量写回表格。
