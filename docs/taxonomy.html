治理策略设计器

调整阈值后复制或下载 JSON，用 apply_governance_policy.py 先 dry-run 再写入 guides/taxonomy.json。刷新 wiki 后 quality、facets、balance 和 coverage 会用同一套策略重算。

应用命令：python3 scripts/apply_governance_policy.py docs --input ~/Downloads/taxonomy_governance_policy.json --write
