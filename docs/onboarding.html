快速开始

#步骤命令
1Open the contributor console
Start from this page to choose the right contribution path.
-
2Batch candidate links before triage
Paste many links, dedupe them against the library and inbox, then export a candidate CSV.
-
3Route a new paper before writing metadata
Use title and abstract to pick an initial research line and taxonomy labels.
-
4Edit or review papers in the dense library
Filter, select, export metadata patches, and manage status workflows.
-
5Run the local quality gate
Use the same checks as CI before opening a PR.
